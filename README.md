# Project Phantom 👻

> Build immersive horror experiences with Python.

Project Phantom is an open-source Python framework for creating interactive horror games and branching narrative experiences. It is designed with modularity, extensibility, and maintainability in mind, making it easy for developers to create unique horror adventures.

Whether you're building a psychological thriller, a survival horror game, or a supernatural mystery, Project Phantom provides the foundation to bring your ideas to life.

---

## ✨ Features

- 📖 Branching story engine
- 💾 Save & load system
- 🎒 Inventory management
- 🎵 Dynamic audio support
- 🎲 Random events
- 🧠 Optional AI-powered storytelling
- 🖥️ Cross-platform desktop support
- 📱 Planned Android support
- 🔌 Modular architecture
- 🌍 Open source

## 🚀 Getting Started

```bash
git clone https://github.com/YOUR_USERNAME/Project_Phantom.git
cd Project_Phantom
```

Create a virtual environment:

```bash
python -m venv .venv
```

Activate it:

**Windows**
```bash
.venv\Scripts\activate
```

**Linux/macOS**
```bash
source .venv/bin/activate
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the project:

```bash
python src/main.py
```
